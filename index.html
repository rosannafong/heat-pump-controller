About

Introduction

The Heat pump Controller is used to control your heat pumps by accessing an online interface using your mobile, tablet or desktop. You just need to set up a Wi-Fi adapter with your heat pump, create an account, and start controlling your heat pump units anywhere and at anytime!

See demo of the Heat pump Conttroller.

Pattern library

A pattern library is available to showcase the building blocks of the user interface for the Heat pump Controller. These patterns has been split into components and features.

Components

Colours
Colour palette for the heat pump control.
Typography
Fonts, headers, paragraph text and links.
Lists
Unordered lists, ordered lists, description lists, and inline lists.
Images
The logo, icons and favicons.
Buttons
Button styles, form buttons, and button groups.
Form components
Text inputs, select boxes, radio buttons, checkboxes, text area, fieldsets, and form validation.
Interactive components
Sliders and switches.
Navigation
Top navigation and tabs.

Features

User login
User sign in and user sign out interfaces.
